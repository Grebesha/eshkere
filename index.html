Виды оружия в игре Counter strike 2

Ножи

Если в начале в CS:GO было всего два вида ножей: для террористов и контр-террористов, то в CS2 ситуация иная. Все скины были перенесены из прошлой части игры, поэтому в CS2 с самого старта достаточно большой выбор ножей.Всего их 20 видов на момент написания материала:

  • Обычный нож / Stock Knife;
  • Штык-нож / Bayonet;
  • Нож-бабочка / Butterfly Knife;
  • Фальшион / Falchion Knife;
  • Складной нож / Flip Knife;
  • Нож с лезвием-крюком / Gut Knife;
  • Охотничий нож / Huntsman Knife;
  • Керамбит / Karambit;
  • Штык-нож M9 / M9 Bayonet.
  • Тычковые ножи / Shadow Daggers;
  • Нож Боуи / Bowie Knife;
  • Медвежий нож / Ursus Knife;
  • Нож «Бродяга» / Nomad Knife;
  • Наваха / Navaja Knife;
  • Стилет / Stiletto Knife;
  • Коготь / Talon Knife;
  • Классический нож / Classic Knife;
  • Скелетный нож / Skeleton Knife;
  • Паракорд-Нож / Paracord Knife;
  • Нож Выживания / Survival Knife.

    Пистолеты

Поэтому выбор оружия на «пистолетку» может оказаться не таким простым как кажется.Несмотря на ранние слухи, оружие в CS2 так и осталось разделенным по сторонам: какие-то пушки можно купить только за сторону спецназа, а с какими-то, наоборот, можно будет поиграть только за террористов

    Пистолеты-пулеметы

    Тут разнообразия меньше, чем у пистолетов, но все равно найдется из чего выбрать. Пистолеты-пулеметы отлично подойдут, если вы хотите сэкономить в раунде, но не хотите играть с одним пистолетом.ПП отлично реализуются во время перестрелок «на бегу» благодаря высокой скорострельности и большому магазину. К сожалению, при этом у них недостаточно высокий урон, чтобы наносить дамаг, сравнимый с более дорогим оружием.

  • PP-Bizon — $ 1400

  • UMP-45 — $ 1200

  • MP5-SD — $ 1500

    Тяжёлое оружие

    Тяжелое вооружение состоит из дробовиков и пулеметов, которые максимально эффективны на близких дистанциях. Такое оружие отлично подойдет, если вы хотите сыграть на закрытой позиции или, например, утаиться в дымовой завесе.Однако на длинной дистанции оно практически бесполезно — разброс пуль, что у дробовиков, что у пулеметов слишком большой. При этом стоимость у оружия не очень высокая, что позволяет ему также находить применение и на форс-раундах.

  • Nova — $ 1050
  • XM1014 — $ 2000
  • MAG-7 — $ 1300(только для спецназа)
  • Sawed-Off — $ 1100 (только для террористов)
  • M249 — $ 5200
  • Negev — $ 1700

    Гранаты

    Гранаты — уникальный вид оружия в Counter-Strike. Все его виды, кроме световой гранаты, имеют только один заряд и больше с собой носить его не получится. Однако гранаты можно передавать тиммейтам, поэтому один игрок может кинуть за раунд больше одной дымовой или зажигательной гранаты.Гранаты в CS2 дают позиционное преимущество и позволяют проводить атаки на точки или же, наоборот, защищаться. С их помощью можно преградить дорогу врагу на некоторое время и подождать пока придет подкрепление. Или же можно заставить обороняющихся уйти с выгодных позиций, после чего заходить на плент.

  • Осколочная граната / HE Grenade — $ 300
  • Световая граната / Flashbang — $ 200
  • Дымовая граната / Smoke Grenade — $ 300
  • Ложная граната / Decoy Grenade — $ 50
  • Коктейль Молотова / Molotov — $ 400 (только для террористов)
  • Зажигательная граната / Incendiary Grenade — $ 600 (только для спецназа)

    Дополнительное снаряжение

    Дополнительное вооружение, которое есть в CS2, это не совсем оружие. Но оно тоже помогает победить, поэтому мы не можем не рассказать про него.В эту категорию входят, например, комплекты брони, а также уникальные для каждого класса вещи типа набора сапера и набора спасения, которые использует только спецназ. Или вовсе уникальный для одного режима медицинский шприц

  • Взрывчатка C4 / C4 Explosive — Бесплатно (только для террористов)
  • Набор сапера / Набор спасения (Defuse Kit / Rescue Kit) — $ 400 (только для спецназа)
  • Шлем + бронежилет / Kevlar + Helmet — $ 1000
  • Бронежилет / Kevlar — $ 650
  • Zeus x27 — $ 200
  • Медицинский шприц / Medi-Shot — бесплатно (только для режимов «Совместный налет», «Запретная Зона» и «Бой насмерть»)
